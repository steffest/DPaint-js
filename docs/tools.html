Tools

Standard Tools

The toolbar is your primary interface for interacting with the canvas. At the very top of the toolbar, you will find Brush Presets. Use these to quickly select a brush shape and size for painting tools.

In Dpaint.js, when you hover over a tool in the toolbar, a small help-text will appear in the bottom right corder, showing the tool name and its keyboard shortcut.

Draw / Brush

The primary painting tool. Use it to draw freehand strokes with the selected brush.

  • Left Click: Draw with the Foreground color.
  • Right Click: Draw with the Background color.
  • ShortKey: B
Select (Rectangle)

Create a rectangular selection.
Selection are mostly used to copy/cut parts of the layer to a new layer.
Use the ESC key to clear a selection

  • Click and drag to define the selection area.
  • Use the arrow keys or the resize handles to transform the selection.
  • ShortKey: S
Note: in Dpaint.js your paint actions are NOT constraint to the selection
Polygon Select

Create a freeform polygonal selection.

  • Click to place points, drag the dots to reposition
  • ESC to exit selection mode
  • ShortKey: P
Flood Select (Magic Wand)

Select a contiguous area of pixels that match the clicked color.

  • Adjust in "Tolerance" in the tool settings to determine how close the surrounding pixels should match.
  • ShortKey: W
Circle / Ellipse

Draw circles or ellipses.

  • Use Shift to constrain to a perfect circle.
  • Click the tool icon again to toggle between "Filled" and "Outline" mode.
  • ShortKey: C
Rectangle / Square

Draw rectangles or squares.

  • Use Shift to constrain to a square.
  • Click the tool icon again to toggle between "Filled" and "Outline" mode.
  • ShortKey: R
Line

Draw straight lines.

  • Use Shift to constrain the line to a 45Â° angle.
  • Use S to toggle "Smooth" (Anti-aliasing) on/off while drawing.
  • ShortKey: L
Curve / Arc

Draw curved lines.

Click and drag to set the start and end points of the arc, then move the mouse to adjust the curve radius.

  • Use S to toggle "Smooth" (Anti-aliasing) on/off while drawing.
  • ShortKey: A
Gradient

Fill the layer with a gradient.

  • Click and drag to define the start and end points of the gradient.
  • Uses the current Foreground and Background colors with an optional Dither pattern.
  • ShortKey: G
Flood Fill

Fill a contiguous area of similar pixels with the foreground color.

  • ShortKey: F
Spray / Airbrush

Spray pixels using the current brush shape.

  • Use the tool settings on the top of the screen to adjust the spray pattern, size and spacing.
  • ShortKey: O
Smudge / Blend

Smear colors on the canvas, like finger painting on wet oil paint.

  • Useful for blending colors together organically.
  • Respects "Palette Lock": will only produce colors from your palette if enabled.
  • Toggle between Smudge/Blend/Sharpen modes using the tool settings on the top of the screen.
  • ShortKey: M
Erase

Erase pixels from the layer using the current brush shape.

  • ShortKey: E
Split View

Toggles the split-screen view, showing the image in 2 panes that you can zoom/pan independently.

  • ShortKey: N
Pan (Hand)

Move the canvas view.

  • Hold Spacebar temporarily to activate the Pan tool while using any other tool.
  • ShortKey: H
Color Picker

Sample a color from the image/canvas to set it as the current Foreground color.

  • Right Click: Set as Background color.
  • Hold Shift while drawing to temporary select this tool.
  • ShortKey: K
Zoom In / Out

Adjust the magnification level of the canvas.

  • You can also use the mousewheel while on the canvas to zoom
  • Or use the zoom buttons top-left of the image to (re)set the zoom.
  • ShortKey: + / -
Transparent color
  • Left click: set the Foreground color to transparent
  • Right click: set the Background color to transparent
Swap Foreground/Background color
  • ShortKey: x

Palette Tools

Please note that while DPaint.js is primarily targeted towards limited color pixel art, it is NOT an indexed color paint program. (Weird, right?!)
This section of the toolbar displays your palette, but you are always free to pick any color outside your palette and use it in your image. The palette is more of a "suggestion" of colors that you want to use, rather than a strict limit.
Of course for low-color pixel art, you want to keep your palette in check, but while working on a piece, it's handy to sometimes stray from your palette.

Palette Editor

Open the palette editor with a large range of options to edit/optimize your palette
Start/Stop Color Cycling

You can define color-cycling ranges in the palette editor

  • ShortKey: TAB
Toggle Color Lock

When color lock is enabled, all the colors in your image will automatically be mapped to your palette.
This enables you to use so called "dirty" tools like smear/blurs/blends/opacity ... without introducing new colors in the image.
Palette Presets

Opens the list of Palette Presets.
You can also extract the palette of an image through the menu "Palette -> From image".

Colors

The bottom part shows your current palette.

  • Left click: select Foreground color
  • Right click: select Background color
  • Double click: open palette editor
  • use the left and right arrow keys to select the previous/next color
  • The active color is indicated with a little white dot

Changing your palette does NOT automatically change the colors in your image if color lock is disabled.
(Remember, DPaint.js is NOT an indexed color program)
If you want to remap your image to the new palette, you can

  • Enable "Color Lock"
  • Use the "Reduce Colors" panel and set the palette to "current". This gives you some additional dither options as well.
  • Change a color in the palette editor with the option "Update layer with color changes" selected.
